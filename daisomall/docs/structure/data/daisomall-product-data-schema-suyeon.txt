products
├─ id (상품 고유값)
├─ name (상품명)
├─ price (가격)
├─ image (대표 이미지)
├─ categoryId (카테고리 연결)
├─ rating (별점)
├─ reviewCount (리뷰 수)
├─ imageLabel (이미지 위 라벨: 1.5만 구매+ 등)
├─ deliveryTypes (오늘배송 / 매장픽업 / 택배배송)
├─ isRecommended (오늘의 추천상품)
├─ isSoldOutSoon (품절임박)
├─ isFiveStar (별점5점)
├─ rank (랭킹)
└─ promotionId (기획전 연결용)
