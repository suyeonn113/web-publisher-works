Tiny Note
모바일 전용 심플 투두 앱 기획

────────────────

프로젝트 개요

Tiny Note는 모바일 전용의 간편 투두/메모 앱이다.

복잡한 분류 없이 사용자가 지금 해야 할 일을 빠르게 적고,
완료 여부만 관리할 수 있는 작은 개인 비서형 앱을 목표로 한다.

메모하듯 가볍게 기록하고,
달력에서 날짜별 할 일을 확인할 수 있도록 설계한다.

────────────────

핵심 방향
모바일 전용 UI
위젯처럼 간결한 화면 구성
빠른 입력 중심
완료 / 미완료만 구분
카테고리 및 복잡한 분류 제거
사용자가 분류 때문에 피로해지지 않도록 설계
“기록” 자체에 집중하는 구조

────────────────

주요 기능

[ 홈 화면 ]

오늘 날짜 표시
오늘의 할 일 목록 표시
빠른 할 일 추가
완료 체크 기능
위젯처럼 단순한 메인 화면 구성

────────────────

[ 미니 달력 ]

월간 달력 표시
날짜별 할 일 존재 여부 표시
미완료 할 일 존재 시 점 또는 아이콘 표시
완료된 할 일만 존재 시 체크 아이콘 표시
날짜 선택 시 해당 날짜 할 일 확인 가능

────────────────

[ 날짜 상세 ]

선택한 날짜의 할 일 목록 표시
완료 / 미완료 구분
자유 텍스트 기반 입력
별도 카테고리 없음

────────────────

데이터 구조

Todo

id
date
content
isDone
createdAt

예시)

{
id: 1,
date: '2026-05-22',
content: '포트폴리오 정리하기',
isDone: false,
createdAt: '2026-05-22T10:00:00'
}

────────────────

포트폴리오 설명 포인트

Tiny Note는 사용자가 할 일을 분류하느라 피로해지지 않도록
완료 / 미완료 중심의 최소 구조로 설계한 모바일 투두 앱이다.

위젯처럼 간결한 UI와 미니 달력을 통해
오늘의 할 일과 날짜별 기록을 빠르게 확인할 수 있도록 구성했다.
